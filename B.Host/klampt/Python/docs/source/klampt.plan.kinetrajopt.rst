klampt.plan.kinetrajopt module
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Summary
=======

A collision-free optimization algorithm, with the main entry point being the
:class:`~klampt.plan.kinetrajopt.kinetrajopt.KineTrajOpt` class.

Importing the ``kinetrajopt`` module puts :class:`~klampt.plan.kinetrajopt.kinetrajopt.KineTrajOpt` and 
:class:`~klampt.plan.kinetrajopt.kinetrajopt.TrajOptSettings` into the module namespace.
